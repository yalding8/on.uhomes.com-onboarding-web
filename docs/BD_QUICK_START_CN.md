# BD 快速上手指南 — uhomes 供应商入驻平台

> **访问地址:** https://on-uhomes-com-onboarding-web.vercel.app

---

## 如何登录

1. 打开 https://on-uhomes-com-onboarding-web.vercel.app/login
2. 输入你的 `@uhomes.com` 邮箱
3. 点击 **Continue with Email**
4. 查看邮箱，获取 8 位数字验证码
5. 输入验证码，点击 **Secure Login**
6. 自动跳转到管理后台

---

## 你能做什么

### 1. 查看你的供应商

登录后，点击左侧栏 **My Suppliers**。可以看到分配给你的所有供应商，包括公司名、邮箱、状态和房源数量。

点击任意供应商可查看详情 — 房源列表、合同信息和负责 BD。

### 2. 邀请新供应商

点击左侧栏 **Invite Supplier**。填写供应商的邮箱和公司名（必填），以及可选的电话、城市和网站。供应商会收到邀请，可以登录开始入驻流程。

### 3. 编辑并发送合同

在供应商详情页，找到状态为 **DRAFT** 的合同，点击 **Edit Contract**。可以填写 9 个合同字段：

- 合作方公司名、联系人、地址、城市、国家
- 佣金比例 (%)
- 合同起止日期
- 覆盖房源

点击 **Save** 保存进度，或点击 **Push for Review** 将合同推送给供应商确认并签署。

### 4. 跟踪合同状态

合同经历以下阶段：

| 状态               | 含义                                      |
| ------------------ | ----------------------------------------- |
| **DRAFT**          | 你正在编辑 — 供应商暂时看不到             |
| **PENDING_REVIEW** | 已推送给供应商 — 等待对方确认             |
| **CONFIRMED**      | 供应商已确认 — 正在创建 DocuSign 签约请求 |
| **SENT**           | 签约邮件已发送给供应商                    |
| **SIGNED**         | 合同已签署 — 完成！                       |

### 5. 重发签约邮件

如果合同处于 **SENT** 状态但供应商未收到邮件，可在供应商详情页点击 **Resend** 重新发送。

## 供应商侧的体验

供应商那边会经历以下流程（供你了解）：

1. **提交申请** — 供应商在公开落地页提交表单（或由你邀请）
2. **登录** — 供应商通过验证码登录，方式和你一样
3. **审核合同** — 供应商看到你推送的合同，点击 **Confirm & Sign** 确认
4. **电子签名** — 供应商收到 DocuSign 邮件，完成电子签名
5. **房源入驻** — 签署完成后，供应商为每个房源填写详细信息

---

## 小贴士

- 你只能看到**分配给你的供应商** — 如需查看其他供应商，请联系管理员
- 合同只能在 **DRAFT** 状态下编辑 — 一旦推送，对你来说变为只读
- 平台支持电脑端和手机端访问
